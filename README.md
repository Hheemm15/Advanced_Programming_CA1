# Advanced Programming CA1

This repository contains my Continuous Assessment for the module **Advanced Programming Techniques**.

## Tasks

### Task 1 – Churros Ordering System (C#)
- Menu-driven console application
- Uses Queue data structure
- Implements OOP concepts (classes, methods, encapsulation)

### Task 2 – Periodic Table (C#)
- Stores first 30 elements
- Uses Dictionary data structure
- Retrieves element by atomic number

### Task 3 – Client-Server Application (Python)
- TCP socket communication
- Stores user data in SQLite database
- Generates unique registration number

### Task 4 – Web Scraping (Python)
- Scrapes book data from website
- Uses BeautifulSoup and requests
- Saves data to CSV file
